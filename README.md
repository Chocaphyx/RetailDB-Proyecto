# RetailDB – Sistema de Base de Datos Empresarial (PostgreSQL + Oracle)

Proyecto académico desarrollado para el curso **Base de datos 2** basado en el caso empresarial *TechStore Perú*.  
El proyecto abarca el ciclo completo de desarrollo de bases de datos: análisis, diseño, implementación, pruebas y evaluación comparativa entre SGBD.

---

## Objetivo
Diseñar e implementar una solución de base de datos empresarial escalable y optimizada, evaluando el rendimiento y viabilidad técnica entre PostgreSQL y Oracle.

---

## Tecnologías utilizadas
- PostgreSQL
- Oracle Database
- SQL / PL-SQL
- Modelado ER
- Normalización (hasta 3FN)

---

## Alcance del Proyecto
El sistema incluye:

- Análisis de requerimientos empresariales
- Modelado conceptual y lógico
- Implementación en dos motores de BD
- Migración de datos
- Pruebas de rendimiento
- Evaluación técnica y económica

---

## Funcionalidades implementadas
- Scripts DDL y DML
- Triggers
- Stored procedures
- Funciones SQL
- Gestión de usuarios y permisos
- Tablespaces y roles (Oracle)
- Consultas complejas optimizadas

---

## Análisis comparativo
Se realizó evaluación entre PostgreSQL y Oracle considerando:

- Rendimiento en consultas
- Uso de recursos
- Seguridad
- Administración
- Costos (TCO a 3 años)

El proyecto incluye recomendaciones técnicas basadas en resultados reales.
